Guía práctica – Clase 24

Actividad:
README, transferencia tecnológica, uso verificable de IA y login pedagógico.

Repositorio:
Repositorio dispuesto por el docente.

Importante:
Seguimos trabajando en el proyecto único de Semana 5.

No debes trabajar directamente en main.
No uses credenciales reales.
No publiques información sensible.

Evidencia:
La evidencia de esta actividad es únicamente el push de tu rama personal a GitHub.

Pasos obligatorios:

1. Entrar al proyecto

cd programadores-para-la-paz-mini-sitio-comunitario

2. Actualizar main local

git checkout main
git pull --ff-only origin main

3. Volver a la rama personal

git checkout nombres_apellidos

4. Incorporar los cambios de main a la rama personal

git merge main

5. Verificar rama actual

git branch

6. Instalar dependencias si hace falta

npm install

7. Levantar el mini-sitio

npm run sitio

8. Abrir en navegador

http://localhost:3000

9. Probar rutas:

http://localhost:3000/api/mensajes
http://localhost:3000/api/calendario
http://localhost:3000/api/resumen

10. Probar login pedagógico

Usuario:
docente

Clave:
demo

11. Probar clave incorrecta

docente

incorrecta

12. Revisar server.js
